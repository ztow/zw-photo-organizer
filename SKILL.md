# 赵兄的智能照片整理 (zw-photo-organizer)

> 一个智能的照片整理工具，支持四级日期识别和自动生成索引。

---

## 🎯 功能特性

- 📁 **智能分类** - 四级日期识别（文件名 → 上级目录 → EXIF → mtime）
- 🔍 **重复检测** - 找出重复/相似照片
- 📊 **自动生成索引** - 整理完成后生成 Markdown 清单
- 🛡️ **安全操作** - 只移动不删除，拿不准的照片单独存放

---

## 🚀 使用方法

### 命令行触发

```bash
/zphoto "/Users/weizhao/Pictures/相机导出图片"
```

### 自然语言触发

```
帮我整理照片
```

> 自然语言触发时会询问目录路径

---

## 📁 目录结构

整理后的结构：

```
{目标目录}/
├── 📁 {年}/
│   └── 📁 {年}_{月}/
│       └── 📁 {年}_{月}_{日}/
│           └── 照片文件
├── 📁 拿不准/              ← 无法确定日期的照片
└── 📄 YYYYMMDD_照片索引.md  ← 自动生成的索引文件
```

---

## 🔍 日期识别优先级

| 优先级 | 方法 | 说明 |
|--------|------|------|
| 1 | 文件名 | `IMG_20241003_143022.jpg` |
| 2 | 上级目录 | `哈尔滨市, 2025年2月12日/` |
| 3 | EXIF | `DateTimeOriginal` |
| 4 | mtime | 文件修改时间（最后手段）|

## ⚙️ 配置说明

配置文件：`config/default.json`

```json
{
  "photo_extensions": [".jpg", ".jpeg", ".png", ".cr2", ".nef", ".arw", ".heic", ".mov", ".mp4"],
  "date_patterns": {
    "filename": ["\\d{4}[_-]?\\d{2}[_-]?\\d{2}"],
    "directory": ["\\d{4}年\\d{1,2}月\\d{1,2}日", "\\d{4}[_-]?\\d{1,2}[_-]?\\d{1,2}"]
  },
  "max_parent_depth": 3,
  "unsure_folder": "拿不准"
}
```

---

## 🛡️ 安全原则

1. **不删除任何照片** - 无论重复、模糊、曝光过度
2. **拿不准的先放一边** - 放入 `拿不准/` 目录
3. **保持原始文件名** - 不修改照片文件本身
4. **原地整理** - 默认在源目录内整理，不复制到新位置
